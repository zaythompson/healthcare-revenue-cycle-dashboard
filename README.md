# Healthcare Revenue Cycle Performance Dashboard

## Overview

This project showcases an interactive Power BI dashboard designed to analyze healthcare revenue cycle performance. The dashboard provides executives with a centralized view of financial and operational KPIs, helping identify revenue trends, claims performance, insurance mix, and outstanding accounts receivable.

## Business Problem

Healthcare organizations require timely visibility into revenue cycle performance to improve collections, monitor claims activity, and reduce outstanding balances. Without centralized reporting, identifying financial trends and operational bottlenecks becomes difficult.

---

## Project Objectives

- Monitor executive revenue cycle KPIs
- Analyze monthly revenue performance
- Compare revenue by insurance type
- Track claims status and collections
- Investigate claim-level payment details
- Demonstrate interactive Power BI drill-through functionality

---

## Key Features

- Executive KPI dashboard
- Revenue trend analysis
- Insurance mix visualization
- Claims status reporting
- Interactive slicers
- Conditional formatting
- Drill-through reporting
- Claim-level detail analysis
